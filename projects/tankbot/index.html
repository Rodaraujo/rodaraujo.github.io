TankBot

A videogame prototype that brings all the 2D Shoot ’em up fun to 3D using Unreal Engine.

TankBot is a 3D video game prototype that looks to imitate the exciting feel of 2D Shoot ’em up games like Contra and Metal Slug. All assets (3D models, sounds, animations, textures, etc.) in the game are placeholders to exemplify the game experience.

Unlike many other third person shooters with cover up mechanics, TankBot rewards forward aggressive play by boosting the player during combat. If you manage to destroy an enemy robot, you’ll get tools to destroy the rest!
